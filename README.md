# 项目介绍
- 本次实验我们设计的是一个商品管理系统，本质上使用C#开发一个供商家和顾客使用的电商平台模型。
- 作为数据库课程的项目设计，我们将数据库SQL语言、视图、权限、存储程序、存储方法等知识应用到实际项目中。
